Simple Honeypot

This is a simple honeypot built in Python. It opens ports 22 (SSH), 443 (HTTP), and 21 (FTP) and logs access attempts. When an access attempt is detected, it sends a message to a Telegram bot with the event information. Additionally, it provides a "graph" command in the Telegram bot to display a graph with all the connections made.

Requirements

  • Python 3.6 or higher
  • Python libraries required: paramiko, http.server, socketserver, python-telegram-bot, requests, configparser, pyftpdlib, sqlite3, matplotlib

Installation

  1. Clone the repository or download the source code.
  2. Install the required libraries by running the following command: 
    pip3 install --user paramiko http.server socketserver python-telegram-bot requests configparser pyftpdlib sqlite3 matplotlib
  3. Create a config.ini file in the same directory as your script and add the following lines, replacing <YOUR_TELEGRAM_BOT_TOKEN> with your Telegram bot token: 
    [Telegram]
token = <YOUR_TELEGRAM_BOT_TOKEN>

Usage

  1. Run the script with the following command: 
    python3 script.py
  2. The script will ask you if you want to open ports 22 (SSH), 443 (HTTP), and 21 (FTP). Type "yes" or "no" to continue or cancel the operation.
  3. Interact with your Telegram bot to receive notifications about access attempts and use the "graph" command to display a graph with all the connections made.

    4. Configuration

    The script requires a configuration file named config.ini with your Telegram bot token. The configuration file should be placed in the same directory as the script.

    Telegram Bot Commands

    • /graph - Displays a graph with all the connections made.

    Notes

    • This simple honeypot is for educational purposes only and should not be used in production environments.
    • Make sure you have the necessary permissions to open the ports on your system and that you are not violating any terms of service of your hosting provider.
